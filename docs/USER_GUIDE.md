## 快速开始

### 首次使用

#### 1. 注册账号

打开应用后,你会看到登录/注册界面:

1. 点击"注册"标签
2. 输入用户名(3-20位字符)
3. 输入邮箱地址
4. 设置密码(至少8位,包含字母和数字)
5. 确认密码
6. 点击"注册"按钮

或者使用快捷登录:
- 点击"微信登录"按钮(需要安装微信)

#### 2. 创建第一个待办

注册成功后:

1. 点击右上角的 ➕ 按钮
2. 输入待办标题(必填)
3. 选择分类(可选)
4. 设置优先级(可选)
5. 设置截止日期(可选)
6. 点击"保存"

恭喜!你已经创建了第一个待办事项。

## 核心功能

### 📝 待办事项管理

#### 创建待办

**方式一:通过应用**
1. 在待办列表页面点击 ➕ 按钮
2. 填写待办信息:
   - **标题**: 简短描述任务(必填)
   - **详细说明**: 更详细的描述(可选)
   - **分类**: 选择所属分类
   - **优先级**: 高/中/低
   - **截止日期**: 设置完成期限
   - **提醒时间**: 设置提醒
   - **标签**: 添加标签便于筛选
3. 点击"保存"

**方式二:通过 Widget**
1. 在主屏幕添加"快速添加"Widget
2. 点击 Widget
3. 输入标题
4. 保存

#### 查看待办

待办列表显示你的所有任务,包括:
- ✅ 完成状态图标
- 📄 任务标题
- 🏷️ 优先级标识(颜色编码)
- 📅 截止日期
- 📊 子任务进度(如有)

**筛选选项**:
- **全部**: 显示所有待办
- **今天**: 今日截止的任务
- **本周**: 本周的任务
- **已完成**: 已完成的任务
- **未完成**: 未完成的任务
- **已逾期**: 过期未完成的任务

**排序选项**:
- 按创建时间
- 按截止日期
- 按标题

#### 编辑待办

1. 点击待办项进入详情页
2. 点击"编辑"按钮
3. 修改信息
4. 点击"保存"

或者:
- 在列表中向左滑动
- 点击"编辑"

#### 完成待办

**方式一**:
- 点击待办前的圆圈图标

**方式二**:
- 进入待办详情
- 点击"标记为完成"按钮

完成后:
- 图标变为 ✅
- 任务标题显示删除线(可选)
- 记录完成时间

#### 删除待办

**方式一**:
- 向左滑动待办项
- 点击"删除"按钮
- 确认删除

**方式二**:
- 进入待办详情
- 点击"删除"按钮
- 确认删除

⚠️ **注意**: 删除操作不可恢复!

### 📂 分类管理

#### 系统预设分类

TodoList 提供 5 个系统预设分类:
- 💼 **工作**: 蓝色,工作相关任务
- 🏠 **生活**: 绿色,日常生活事务
- 📚 **学习**: 橙色,学习和培训
- ❤️ **健康**: 红色,健康和运动
- 🎯 **目标**: 紫色,长期目标

系统分类不能删除或重命名,但可以修改颜色和图标。

#### 创建自定义分类

1. 进入"分类"标签
2. 点击 ➕ 按钮
3. 输入分类名称
4. 选择图标(从 SF Symbols 中选择)
5. 选择颜色(8种预设颜色)
6. 点击"保存"

#### 编辑分类

1. 在分类列表点击要编辑的分类
2. 修改名称、图标或颜色
3. 点击"保存"

#### 删除分类

1. 向左滑动分类项
2. 点击"删除"
3. 确认删除

⚠️ **注意**:
- 系统预设分类不能删除
- 删除分类后,该分类下的待办会变为"未分类"

#### 查看分类统计

每个分类显示:
- 总任务数
- 未完成任务数
- 今日任务数
- 完成进度条

### 🍅 番茄钟功能

番茄工作法是一种时间管理方法,帮助你保持专注。

#### 番茄钟原理

- **工作时间**: 25 分钟专注工作
- **短休息**: 5 分钟休息
- **长休息**: 完成 4 个番茄钟后,休息 15 分钟

#### 开始番茄钟

**方式一:独立使用**
1. 进入"番茄钟"标签
2. 点击"开始"按钮
3. 专注工作 25 分钟
4. 时间到后,自动提醒休息

**方式二:关联任务**
1. 在待办详情页点击"开始番茄钟"
2. 或在番茄钟页面选择任务
3. 开始计时
4. 番茄钟数会自动记录到任务中

#### 番茄钟控制

- **暂停**: 点击"暂停"按钮暂时停止计时
- **继续**: 点击"继续"恢复计时
- **跳过**: 点击"跳过"结束当前阶段
- **重置**: 点击"重置"清零计时器

#### 自定义设置

1. 进入"个人中心" → "番茄钟设置"
2. 可自定义:
   - 工作时长(15-60分钟)
   - 短休息时长(3-15分钟)
   - 长休息时长(10-30分钟)
   - 长休息间隔(2-8个番茄钟)
   - 自动开始选项
   - 声音提醒
   - 通知提醒

#### 番茄钟统计

在统计页面查看:
- 今日完成的番茄钟数
- 本周番茄钟趋势
- 每个任务的番茄钟投入

### 🧩 Widget 小组件

Widget 让你无需打开应用即可查看和管理待办。

#### 添加 Widget

1. 长按主屏幕空白处
2. 点击左上角 ➕
3. 搜索"TodoList"
4. 选择 Widget 尺寸
5. 点击"添加小组件"

#### Widget 尺寸和功能

**小尺寸 Widget**:
- 显示今日待办数量
- 显示未完成任务数
- 点击打开应用

**中尺寸 Widget**:
- 显示最多 4 个待办
- 显示优先级和截止日期
- 点击任务查看详情

**大尺寸 Widget**:
- 显示最多 8 个待办
- 完整的任务信息
- 分优先级分组显示

**快速添加 Widget**:
- 中尺寸
- 点击快速添加新任务
- 输入标题即可创建

#### Widget 刷新

Widget 数据会自动刷新:
- 每 15 分钟更新一次
- 应用内修改数据时立即刷新
- 手动刷新:进入应用后自动触发

## 高级功能

### 📋 子任务

将大任务拆分为多个小步骤。

#### 添加子任务

1. 进入待办详情页
2. 滚动到"子任务"部分
3. 点击"添加子任务"
4. 输入子任务标题
5. 保存

#### 管理子任务

- **完成**: 点击子任务前的圆圈
- **编辑**: 点击子任务文本
- **删除**: 向左滑动删除
- **排序**: 长按拖动调整顺序

#### 查看进度

- 详情页显示子任务进度
- 格式: "已完成/总数" (如 3/5)
- 进度条直观显示

---

### 🏷️ 标签系统

使用标签灵活组织任务。

#### 添加标签

1. 创建或编辑待办时
2. 在"标签"字段输入
3. 多个标签用逗号分隔
4. 如: `紧急,重要,会议`

#### 按标签筛选

1. 在待办列表点击筛选按钮
2. 选择标签
3. 只显示包含该标签的任务

#### 常用标签建议

- `紧急`: 需要立即处理
- `重要`: 优先级高但不紧急
- `等待`: 等待他人反馈
- `会议`: 会议相关
- `外出`: 需要外出处理
- `电话`: 需要电话沟通

---

### ⏰ 提醒功能

设置提醒,不错过重要任务。

#### 设置提醒

1. 创建或编辑待办
2. 开启"提醒"开关
3. 选择提醒时间
4. 保存

提醒时间可以:
- 与截止日期相同
- 提前 10 分钟
- 提前 1 小时
- 提前 1 天
- 自定义时间

#### 提醒通知

到达提醒时间时:
- 推送通知
- 显示任务标题和详情
- 点击通知直达任务详情

⚠️ **首次使用需授权通知权限**

#### 管理提醒

- 查看所有提醒:在提醒列表
- 取消提醒:关闭提醒开关
- 延后提醒:修改提醒时间

## 使用技巧

### 🎯 提高效率的技巧

#### 1. 合理使用优先级

- **高优先级**: 重要且紧急,今天必须完成
- **中优先级**: 重要但不紧急,本周完成
- **低优先级**: 不紧急,有时间再做

#### 2. 每日规划

每天早上:
1. 查看"今天"筛选项
2. 评估今日任务
3. 调整优先级
4. 设置 3-5 个主要目标

#### 3. 番茄工作法最佳实践

- 开始前准备好所需资料
- 番茄钟期间避免打断
- 休息时真正放松
- 记录完成的番茄钟数

#### 4. 分类使用建议

- 不要创建太多分类(5-8个即可)
- 按工作性质分类,而非项目
- 定期检查分类合理性

#### 5. 子任务运用

何时使用子任务:
- 任务较大,需要多个步骤
- 需要跟踪进度
- 需要拆分为多天完成

#### 6. 标签组织

建立标签体系:
- 按紧急程度:`紧急`,`一般`
- 按地点:`家`,`公司`,`外出`
- 按时长:`5分钟`,`半小时`,`1小时`
- 按项目:`项目A`,`项目B`

#### 7. 周回顾

每周末:
1. 查看本周完成情况
2. 整理未完成任务
3. 规划下周重点
4. 清理不需要的待办

## 常见问题

### 账号相关

**Q: 忘记密码怎么办?**
A: 在登录页面点击"忘记密码",通过邮箱重置。

**Q: 可以修改用户名吗?**
A: 用户名注册后不可修改,但可以修改显示昵称。

**Q: 如何切换账号?**
A: 进入"个人中心" → "登出",然后用新账号登录。

**Q: 数据可以同步吗?**
A: 当前版本数据存储在本地。未来版本将支持 iCloud 同步。

---

### 功能使用

**Q: 如何批量删除已完成任务?**
A: 进入"设置" → "数据管理" → "清除已完成任务"。

**Q: 可以恢复已删除的待办吗?**
A: 目前不支持恢复,请谨慎删除。建议定期导出数据备份。

**Q: 为什么有的任务显示为逾期?**
A: 未完成且截止日期已过的任务会标记为逾期,以红色显示。

**Q: 如何隐藏已完成的任务?**
A: 点击筛选按钮,选择"未完成"即可。

---

### Widget 相关

**Q: Widget 数据不更新?**
A:
1. 检查 Widget 是否正确配置
2. 打开应用查看是否有数据
3. 尝试移除 Widget 重新添加
4. 重启设备

**Q: Widget 显示"无待办"但应用里有?**
A:
1. 检查 Widget 显示的是否为特定筛选
2. 小尺寸 Widget 只显示今日任务
3. 尝试手动刷新

**Q: 快速添加 Widget 不工作?**
A:
1. 确认已授予通知权限
2. 更新到最新版本
3. 联系技术支持

---

### 番茄钟相关

**Q: 番茄钟可以自定义时长吗?**
A: 可以,在"设置" → "番茄钟设置"中自定义各阶段时长。

**Q: 番茄钟期间可以暂停吗?**
A: 可以暂停,但不建议。番茄工作法强调专注和不被打断。

**Q: 番茄钟到时没有提醒?**
A: 检查:
1. 是否开启了通知权限
2. 设置中是否开启提醒
3. 手机是否静音

---

### 性能和故障

**Q: 应用运行卡顿?**
A:
1. 重启应用
2. 清理已完成任务
3. 更新到最新版本
4. 重启设备

**Q: 数据丢失了?**
A:
1. 检查是否误删
2. 检查筛选条件
3. 恢复备份数据

**Q: 应用崩溃?**
A:
1. 更新到最新版本
2. 重启设备
3. 联系技术支持并提供崩溃日志

---

### 数据管理

**Q: 如何导出数据?**
A: "设置" → "数据管理" → "导出数据",选择格式(CSV/JSON)。

**Q: 可以导入其他应用的数据吗?**
A: 支持导入 CSV 格式的任务列表。

**Q: 如何备份数据?**
A:
1. 定期导出数据
2. 使用 iCloud 备份设备(包含应用数据)
