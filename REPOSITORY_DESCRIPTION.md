## 完整描述（Full Description）
适用于 README 顶部或项目介绍：

```
📝 TodoList - 现代化的待办事项管理应用

一款基于 SwiftUI 和 SwiftData 构建的功能强大、界面优雅的 iOS 待办事项管理应用。
采用最新的 iOS 开发技术栈，提供流畅的用户体验和丰富的功能。

🌟 核心功能
• 完整的待办事项管理（创建、编辑、删除、完成）
• 子任务支持，将大任务拆分为小任务
• 自定义分类系统（图标、颜色、名称）
• 番茄钟计时器，提升工作效率
• 日历视图，直观管理每日任务
• 数据统计分析，追踪完成趋势
• 主屏幕 Widget 小组件支持
• 本地通知提醒，不错过重要事项
• 深色模式支持

🛠️ 技术栈
• SwiftUI - 声明式 UI 框架
• SwiftData - 现代化数据持久化
• MVVM 架构模式
• WidgetKit - 小组件支持
• UserNotifications - 本地通知
• Combine - 响应式编程

✨ 适用于学习和参考
• 完整的 SwiftUI 项目结构
• SwiftData 最佳实践
• Widget 开发示例
• 权限管理实现
• 通知系统集成
```
